GIF Palette Counter
	Version 1.01
_____________________


=    Overview    =

This program will read in all the GIF images in a folder and generate a report detailing which ones share the same palette.



=   Installing   =

Unzip the program to the directory of your choice.



=   How to use   =

Simply run the program, press the "Open input folder" button, and select a folder containing GIF images.
After the folder has been processed, the program will offer to save a report.  You can cancel this if you do not wish to save a report.
In either case, the information will also be displayed in the program's interface.gingDisabled=1 to the [Settings] section of 3ds2bin.ini.
